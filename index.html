Project Goals

The goal for this project was to strengthen my knowledge of JavaScript and Node.js by building a simple, web-based game.


Technologies Used

  • HTML/CSS
  • Canvas
  • JavaScript
  • jQuery
  • Node.js
  • Express
  • Socket.io

Design

The core game is written entirely in JavaScript. I used Node.js to run the server along with Express and Socket.io to serve files and handle communication between the client and the server.


The game is displayed using Canvas, and I use HTML, CSS, and jQuery to handle user interaction, menus, and popups.


The game features a simple lobby that allows users to log in and play against a specific opponent. Games are saved automatically and users can reload a game in the event of a disconnect.


In the future, I would like to incorporate authenticated login and a database to store user information and saved games. I would also like to add the ability to chat and observe active games.


Zombie Chase +-

Restart

Project Goals

The goal for this project was to experiment with Processing by building a simple simulation.


Technologies Used

  • Processing
  • Processing.js

Design

The zombie simulation generates a number of 'persons', most of which begin as normal while a few are infected. Normal persons move away from the infected, and the infected move toward the normal. As soon as they touch, the infection spreads.


By adjusting the amount of random movement and the speed of the actors, you can observe different effects on the rate of spread.


The processing file is then embedded into the page using Processing.js. This library allows for easy visualization as well as interaction between JavaScript and Processing.


While this example is quite simplistic, it demonstrates a powerful way to incorporate complex data simulations and visualizations into dynamic web applications.

Undergraduate Research +-

Computational Drug Discovery

My undergraduate research in the lab of Dr. David Bevan focused on using computer simulations for drug discovery. In particular, we were interested in the protein monoamine oxidase B (MAOB). MAOB is an enzyme that functions in the oxidation of neurotransmitters and other small molecules, and as a byproduct, this process increases oxidative stress in the brain. This action has been implicated in Parkinson’s disease, and several MAOB inhibitors have been developed that slow progression of the disease.


Unfortunately, many of these drugs suffer from a lack of specificity, resulting in unwanted side effects. New drugs with more specific binding could reduce these side effects and improve drug efficacy. To this end, we developed and validated a method for screening new drugs in silico.


By optimizing parameters for drug binding, we were able to validate out method using a panel of drugs with known binding affinities. This provides a foundation for high-throughput drug screening efforts. We also identified key residues within the enzyme that contribute to drug binding. This knowledge can be used for rational drug design as an alternative to high-throughput screening.

  • Bevan Lab at Virginia Tech
  • Harkcom, W.T. and Bevan, D.R. (2007) "Molecular docking of inhibitors into monoamine oxidase B" Biochem. Biophys. Res. Commun. 360 (2): 401-406.
