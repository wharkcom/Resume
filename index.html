Ultimate Tic-Tac-Toe +-

Play Game

Project Goals

The goal for this project was to strengthen my knowledge of JavaScript and Node.js by building a simple, web-based game.


I chose Ultimate Tic-Tac-Toe so that I could focus on **fundamentals** rather than complex game logic.


This project demonstrates my ability to develop a complex web application.


Technologies Used

  • HTML/CSS
  • Canvas
  • JavaScript
  • jQuery
  • Node.js
  • Express
  • Socket.io

Design

The core game is written entirely in JavaScript. I used Node.js to run the server along with Express and Socket.io to serve files and handle communication between the client and the server.


The game is displayed using Canvas, and I use HTML, CSS, and jQuery to handle user interaction, menus, and popups.


The game features a simple lobby that allows users to log in and play against a specific opponent. Games are saved automatically and users can reload a game in the event of a disconnect.


In the future, I would like to incorporate authenticated login and a database to store user information and saved games. I would also like to add the ability to chat and observe active games.
